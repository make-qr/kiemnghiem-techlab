# NATEKVN - Dịch vụ Kiểm nghiệm và Chứng nhận

Đây là mã nguồn cho trang web landing page của dự án kiemnghiem.natekvn.com, cung cấp thông tin về các dịch vụ kiểm nghiệm và chứng nhận.

## Cấu trúc dự án

```
kiemnghiem.natekvn.com/
├── index.html              # Trang chủ
├── css/
│   ├── style.css           # CSS chính
│   └── service-page.css    # CSS cho trang dịch vụ
├── js/
│   └── main.js             # JavaScript chính
├── images/                 # Thư mục chứa hình ảnh
├── pages/                  # Các trang dịch vụ
│   ├── kiem-nghiem-thuc-pham.html
│   ├── chung-nhan-vietgap.html
│   └── ...
└── README.md
```

## Tính năng

- Thiết kế responsive, tương thích với nhiều thiết bị
- Sử dụng HTML, CSS, và JavaScript thuần
- Phong cách thiết kế hiện đại với màu chủ đạo là xanh nước biển
- Nội dung đầy đủ cho các dịch vụ kiểm nghiệm và chứng nhận

## Các trang landing page

1. **Kiểm nghiệm:**
   - Kiểm nghiệm thực phẩm
   - Kiểm nghiệm dược phẩm
   - Kiểm nghiệm mỹ phẩm
   - Kiểm nghiệm thức ăn chăn nuôi
   - Kiểm nghiệm thành phần dinh dưỡng
   - Kiểm nghiệm nước giải khát
   - Kiểm nghiệm bánh kẹo

2. **Chứng nhận:**
   - Chứng nhận VietGAP
   - Chứng nhận Organic
   - Chứng nhận Halal
   - Chứng nhận Global GAP
   - Chứng nhận SMETA SEDEX
   - Quan trắc môi trường
   - Kiểm nghiệm khí thải
   - Tư vấn môi trường

## Cách sử dụng

1. Clone repository này
2. Mở file `index.html` trong trình duyệt web để xem trang chủ
3. Điều hướng đến các trang dịch vụ chi tiết thông qua liên kết từ trang chủ

## Hình ảnh

Để hoàn thiện trang web, bạn cần thêm các hình ảnh sau vào thư mục `images/`:

- `hero-bg.jpg` - Hình nền cho phần hero của trang chủ
- `service-hero-bg.jpg` - Hình nền cho phần hero của trang dịch vụ
- `cta-bg.jpg` - Hình nền cho phần kêu gọi hành động
- `food-testing.jpg` - Hình ảnh cho trang kiểm nghiệm thực phẩm
- `vietgap-certificate.jpg` - Hình ảnh cho trang chứng nhận VietGAP
- Các hình ảnh khác cho các trang dịch vụ còn lại
